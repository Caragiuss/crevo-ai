Crevo AI is a creator dashboard that helps track content performance, improve engagement, and generate AI-powered content ideas.

## Features
- Account health score based on posting frequency & engagement
- Platform filtering (Instagram, TikTok, Facebook)
- AI-generated post ideas & captions
- Performance analytics with charts
